# Netflix Movies & TV Shows Data Analysis

## Project Overview
This project analyzes the Netflix Movies and TV Shows dataset using Python, Pandas, Matplotlib, and Seaborn. The objective is to clean the data, perform exploratory data analysis (EDA), create visualizations, and derive meaningful insights.

## Dataset
Netflix Movies & TV Shows Dataset (Kaggle)

## Libraries Used
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Steps Performed
1. Loaded and inspected the dataset.
2. Cleaned missing values and duplicates.
3. Performed exploratory data analysis.
4. Created six different visualizations.
5. Generated business insights from the analysis.

## Visualizations
- Bar Chart
- Line Chart
- Histogram
- Scatter Plot
- Pie Chart
- Heatmap

## Key Insights
- Movies are more numerous than TV Shows.
- The United States has the highest number of Netflix titles.
- Most titles were released after 2010.
- Netflix's content library has grown rapidly over recent years.
- Mature audience content (TV-MA) is the most common rating.
